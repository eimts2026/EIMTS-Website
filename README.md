# Emerald Isle Manpower

A multi-page recruitment website built with React, TypeScript and Vite.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production output is generated in `dist/`. Each public route receives its
own SEO-ready `index.html`, while client-side navigation keeps page changes
fast.

## Vercel

Import the repository into Vercel. The included `vercel.json` uses
`npm run build` and publishes the `dist` directory.
